# Miscellaneous analysis tools
## Content:
* scrambleseq: Python script to generate scrambled DNA or RNA sequences (permutations with the same base content) from an input sequence and evaluate their thermodynamic stability using the external command-line tool **RNAfold** (ViennaRNA package) [1].
